# API 接口文档

## 快速开始

1. 先读 [COMMON.md](COMMON.md) 了解通用约定（响应结构、分页、签名等）
2. 按需查看对应模块文档获取详细接口说明
3. 对接单个模块时，只需 **通用约定 + 该模块文档** 即可
